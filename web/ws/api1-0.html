API v1.0 Functions

Login
Description - Takes 'username' and 'password' and returns a session token if successful
username - The users username
password - The users password
Usage - isenseproject.org/ws/api.php?method=login&username=XXX&password=XXX
Return - {"status":200,"data":{"uid":"XXX","session":"XXXXXXXXXXX"}}

getExperiment
Description - Takes 'experiment' and returns the meta data associated with it
experiment = The id of the desired experiment.
Usage - isenseproject.org/ws/api.php?method=getExperiment&experiment=1

getExperiments
Description - Takes 'page', 'limit', 'query', 'sort' and returns a list of experiments with their meta data
page = The page of size limit, offset into the full list of results, default 1
limit = The number of results desired, default 10
query = Search terms by tags/name of experiment, default "", example "water"
sort = What to sort results by. accepts recent, popularity, activity, rating
Usage - isenseproject.org/ws/api.php?method=getExperiments

getPeople
Description - Takes 'page','limit','status' and returns a list of people with their meta data
page = The page of size limit, offset into the full list of results, default 1
limit = The number of results desired, default 10
query = Search terms by names of people, default "", example "Bob"
Usage - isenseproject.org/ws/api.php?method=getPeople

getSessions
Description - Takes 'experiment' and returns a list of sessions associated with that experiment and their meta data
experiment = The id of the desired experiment.
Usage - isenseproject.org/ws/api.php?method=getSessions&experiment=1

getExperimentFields

getUserProfile
Description - Takes 'experiment' and returns a list of images associated with that experiment
Requires - session_key, user
session_key = The session token returned by a valid login.
user = The id of a user.
Usage - isenseproject.org/ws/api.php?method=getExperimentImages&experiment=1

getExperimentByUser

user = The id of a user.
Usage - isenseproject.org/ws/api.php?method=getExperimentByUser&user=1

createSession
Description - Takes 'session_key','eid','description','name','street','city','state'
Requires - session_key, eid
session_key = The session token returned by a valid login.
eid = The id of the desired experiment.
description = The text description of the session.
name = The name of the session.
city/street/state = The parts of the address where the session was created. Usage - isenseproject.org/ws/api.php?method=createSession&eid=1&session_key=XXXXXXXXXXX
